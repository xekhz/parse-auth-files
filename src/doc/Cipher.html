    Recebe uma string e devolve cifrada Utiliza para gerar a cifra o aes-256-cbc e cifra com o a chave solicitada na criacao do pbjecto

    # File Cipher.rb, line 20
def cipher(message)
  @cipher = OpenSSL::Cipher::Cipher.new("aes-256-cbc")
  @cipher.encrypt
  @key = Digest::SHA1.hexdigest("#{@password}")
  @cipher.key = @key
  encrypted = @cipher.update(message)
  encrypted << @cipher.final
  return encrypted
end

def decipher(enc_message)
  @cipher = OpenSSL::Cipher::Cipher.new("aes-256-cbc")
  @cipher.decrypt
  @key = Digest::SHA1.hexdigest("#{@password}")
  @cipher.key = @key
  decrypted = @cipher.update(enc_message) + @cipher.final
  #decrypted << @cipher.final
  return decrypted
end
